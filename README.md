# BatchScriptGenerator
Batch script generator written on C#

# Features:
* **Large set of commands**
* **Simple interface, working in cmd**
* **Import/Export batch files as `.bsgbuildfile` and build scripts by them**

> [!IMPORTANT]
> Current version **BatchScriptGenerator** was created on **.NET Framework 4.7.2**.
> I am working to redesing BSG to newest version of .NET Framework

# How to install
There are **two** ways how to install BSG

## 1. Download from repository
* Go to releases  
* Download BSG from the last release
* Run downloaded file

## 2. Clone repository from GitHub
* Clone repository to Your computer
* Run `Program.cs` through Visual Studio
